# Wordle

A clean and modern Wordle-style game built with **HTML**, **SCSS**, and **TypeScript**.  
Players get **6 guesses** to find the hidden word, with real-time visual feedback and a built-in leaderboard to track top scores.

---

## Features
- 6 guesses per game  
- Color-coded feedback for correct, present, and missing letters  
- Leaderboard for tracking player performance  
- Responsive and accessible UI built with SCSS  
- Fast and structured game logic powered by TypeScript  

---

## Tech Stack
- HTML  
- SCSS  
- TypeScript
